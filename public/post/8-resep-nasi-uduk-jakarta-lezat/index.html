  1. beras 1/2 liter
  2. santan liter santan kental santan cair 750 liter

Langkah-langkah Untuk Menyiapkan Nasi uduk jakarta:

  1. Cuci beras. Panaskan kukusan, masukan beras setelah air mendidih.
  2. Kukus beras selama 15-20 menit tergantung api yang di gunakan, saya menggunakan api besar.
  3. Setelah beras di kukus selama 20 menit, panaskan santan dan semua bahan lainnya dengan api kecil hingga mendidih. Jangan lupa di aduk terus menerus.
  4. Setelah santan mendidih, matikan api lalu pindahkan semua beras kedalam air santan, dan rendam selama 15 menit sampai santan meresap semua kedalam beras.
  5. Panaskan kukusan, setelah mendidih kukus beras yang telah di rendam santan. Jangan lupa di aduk sesekali jangan terlalu sering. Masak hingga beras menjadi nasi.
